# 📋 Aplikasi To-Do List Persisten

Aplikasi daftar tugas sederhana yang menyimpan data secara lokal menggunakan AsyncStorage, sehingga data tetap ada meskipun aplikasi ditutup.

## ✨ Fitur
### 🟢 Level 1 (Wajib)
- Tambah tugas baru
- Tampilkan daftar tugas
- Hapus tugas
- Simpan data ke AsyncStorage
- Pesan kosong jika daftar belum ada
- Data tetap tersimpan setelah aplikasi ditutup

### 🟡 Level 2 (Tambahan)
- ✏️ **Tandai tugas selesai** (teks dicoret)
- 🔎 **Cari & filter tugas** berdasarkan kata kunci
- ⚠️ Konfirmasi sebelum menghapus

## 🛠️ Teknologi yang Digunakan
- React Native
- Expo
- AsyncStorage

## 🚀 Cara Menjalankan
1. Clone repositori ini
2. Jalankan `npm install`
3. Jalankan `npx expo start`
4. Scan QR Code dengan aplikasi Expo Go
